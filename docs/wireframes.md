# Wireframes — my-first-web

깔끔하고 미니멀한 개인 블로그 기준의 초안 와이어프레임입니다.  
shadcn/ui 컴포넌트를 기준으로 배치했습니다.

## 1) 홈 페이지

### 목적
- 블로그 소개
- 포스트 카드 목록 노출
- 검색바로 글 찾기

### 와이어프레임

```text
┌──────────────────────────────────────────────────────────────┐
│  내 블로그                          [로그인] [회원가입]      │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  읽기 편한 속도로 기록을 쌓는 개인 블로그                    │
│  하루의 생각, 배운 점, 짧은 메모를 부담 없이 남기는 공간     │
│                                                              │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  제목 또는 내용 검색                         [검색]     │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  [글 목록 보기]   [새 글 쓰기]                                │
│                                                              │
│  ┌────────────────┐  ┌────────────────┐  ┌────────────────┐  │
│  │  Post Card     │  │  Post Card     │  │  Post Card     │  │
│  │  제목          │  │  제목          │  │  제목          │  │
│  │  본문 미리보기 │  │  본문 미리보기 │  │  본문 미리보기 │  │
│  │  작성자 / 날짜  │  │  작성자 / 날짜  │  │  작성자 / 날짜  │  │
│  │      [글 읽기]  │  │      [글 읽기]  │  │      [글 읽기]  │  │
│  └────────────────┘  └────────────────┘  └────────────────┘  │
│                                                              │
│  ┌────────────────┐  ┌────────────────┐                      │
│  │  Post Card     │  │  Post Card     │                      │
│  └────────────────┘  └────────────────┘                      │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### shadcn/ui 사용 위치
- `Card`: 포스트 카드, 소개 영역
- `Button`: 글 목록 보기, 새 글 쓰기, 글 읽기, 검색
- `Input`: 검색바

---

## 2) 포스트 작성 페이지

### 목적
- 제목과 본문을 입력
- 저장 버튼으로 새 글 작성

### 와이어프레임

```text
┌──────────────────────────────────────────────────────────────┐
│  ← 목록으로 돌아가기                                          │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  새 글 작성                                                  │
│  제목과 내용을 차분하게 작성해 보세요.                       │
│                                                              │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  제목                                                  │  │
│  │  [ 제목을 입력하세요                               ]   │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  내용                                                  │  │
│  │  ┌──────────────────────────────────────────────────┐  │  │
│  │  │                                                  │  │  │
│  │  │  여러 줄 내용을 입력하는 영역                    │  │  │
│  │  │                                                  │  │  │
│  │  └──────────────────────────────────────────────────┘  │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  [게시글 저장하기]   [취소]                                  │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### shadcn/ui 사용 위치
- `Card`: 작성 폼 전체
- `Input`: 제목 입력
- `Textarea`: 내용 입력
- `Button`: 저장, 취소, 목록으로 돌아가기

---

## 디자인 메모
- 배경은 밝게 유지한다.
- 글은 카드 안에 넣어서 읽기 쉽게 만든다.
- 버튼은 2~3개만 보여서 복잡하지 않게 한다.
- 모바일에서는 카드가 세로로 쌓이도록 만든다.
